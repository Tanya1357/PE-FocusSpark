# PE - FocusCat 🐱⏱️

**Reaper计时猫咪伴侣脚本**

一个番茄钟风格的工作计时器，配有可爱的猫咪陪伴，提供完成反馈、连击奖励和加班预警。

## 功能特性

- ⏱️ **灵活计时器**：10/20/30分钟预设 + 自定义时间
- 🔔 **闹钟提醒**：倒计时结束时音效和视觉提示
- 🐱 **猫咪伙伴**：像素风格猫咪陪伴，状态随工作变化
- 🎉 **完成反馈**：点击完成触发动画和音效
- 🎮 **连击系统**：3/5/10连击触发特殊效果
- 🎵 **自定义音效**：支持导入WAV/MP3/OGG音频
- 📊 **工作统计**：今日完成数、总耗时、按时/超时统计
- ⚠️ **加班预警**：19:30后猫咪愤怒，提醒该休息了

## 使用方法

### 基本工作流

1. **评估任务** → 选择时间预设（如20分钟）
2. **开始计时** → 点击"开始"，倒计时开始
3. **专注工作** → 猫咪进入工作状态，陪你专注
4. **时间到** → 闹钟响起，猫咪提醒你
5. **点击完成** → 触发奖励动画和音效
6. **重复** → 继续下一个任务

### 快捷键（可在Reaper Actions中自定义）

- **F1** - 开始计时
- **F2** - 暂停/继续
- **F3** - 完成（Done）
- **F4** - 停止计时

### 自定义音效

1. 点击"音效设置"按钮
2. 选择要替换的事件（完成/连击/闹钟/警告）
3. 点击"浏览"选择音频文件（WAV/MP3/OGG）
4. 保存设置

**提示**：可以导入游戏音效（如瓦罗兰特的"ACE!"）作为5连击奖励！

## 猫咪状态说明

- **IDLE** 😺：闲置趴着，尾巴缓慢摆动
- **WORKING** 💼：工作中，眼睛专注，尾巴有节奏摆动
- **HAPPY** 😸：完成时开心跳跃，爱心飞舞
- **ALARM** ⏰：闹钟时跳起提醒
- **ANGRY** 😾：加班时背对用户，尾巴用力拍打

## 连击奖励

- **3连击** 🎆：小烟花效果
- **5连击 (ACE)** 😎：猫咪戴墨镜，霓虹光效
- **10连击** 💥：超级烟花，屏幕震动

## 数据持久化

所有数据自动保存到 `data.json`：
- 今日统计（每天0点自动重置）
- 自定义音效设置
- Combo最高记录
