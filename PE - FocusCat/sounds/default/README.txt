-- combo3.wav      - 3连击音效（开心的呼噜声）
-- combo5.wav      - 5连击/ACE音效（兴奋的多声喵）
-- combo10.wav     - 10连击音效（超级兴奋）
-- alarm.wav       - 闹钟音效（猫叫钟声）

-- 支持格式：WAV, MP3, OGG
-- 
-- 如果文件不存在，脚本将不播放音效（静音模式）
